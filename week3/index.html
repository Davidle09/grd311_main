What is a computer directory structure?

A computer directory structure or file system organizes and stores files in a computer's hard drive which creates a hierarchicial structure that lets users locate and access files and folders.

What is the root directory?

A root directory is a top-level directory which contains files and folders on a computer. These can contain sub-directories which can contain more sub-directories, creating a tree-like structure.

What is the purpose of subdirectories?

Subdirectories are created as folders to organize content by type such as img for image files, css for css files and js for javascript files, and pages for HTML files.

What is a file path?

A file path is the location of a file in the computer's file system.

Why is the directory structure essential for organizing and managing files on a computer?

The directory structure is essential for organizing and managing files on a computer because it provides an efficient way to find and access data.

What is HTML?

HTML stands for Hyper Text Markup Language.

What is the purpose of HTML?

HTML is used to display documents designed for web browsers and is the standard markup language for displaying these documents.

What is the difference between HTML and CSS?

HTML is a tag based language used to define elements in a document by embedding codes or tags using common start of stop characters. CSS is used to style HTML elements in a web page.

How create a site definition in Dreamweaver

  1. Open Dreamweaver
  2. Go to site and click new site
  3. Change site name
  4. Change the local site folder by clicking on the folder icon
  5. Create new folder inside documents and click create
  6. Click choose
  7. Click save

How to create a basic site structure

  1. Go to the files tab
  2. Right click on the site definition
  3. Create new folders with the names "img" for image files, "css" for css, and "js" for javascript
  4. Create a new file with the name Index.html

How to add an image to a single page

  1. Go to the insert tab
  2. Click image
  3. Go to img directory of the site directory
  4. Select the image and open

How to create an external link to another page

  5. Change target to new
